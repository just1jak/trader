# STEM with Roo Plan

## 1. Roo's Voice
- **Tone**: Friendly, curious, encouraging, slightly playful.
- **Style**: Uses simple analogies, short sentences, occasional sound effects (e.g., "Zap!", "Whirr!").
- **Vocabulary**: Age-appropriate (6-10 years), avoids jargon unless explained.
- **Catchphrase**: "Let's find out how it works!"
- **Consistency**: Always ends segments with a question to engage kids.

## 2. First Post Using Remotion
- **Goal**: 15-second animated intro showing Roo explaining a basic electronics concept (e.g., what is a circuit?).
- **Scene Breakdown**:
  1. **0-3s**: Roo appears, waves, says "Hi! I'm Roo!".
  2. **3-6s**: Animated battery and wire appear; Roo points: "This is a battery. It gives energy."
  3. **6-9s**: Light bulb lights up when circuit completes; Roo: "When the wire connects, the light turns on!"
  4. **9-12s**: Roo holds a switch: "Add a switch to turn it off and on."
  5. **12-15s**: Roo gives thumbs up: "Try making your own circuit!" + show text: "Next: Build a flashlight!"
- **Technical**:
  - Use Remotion to compose video from React components.
  - Assets: Simple SVG characters, bright colors.
  - Export as MP4, upload to Slack channel #stem-with-roo with caption.

## 3. First Book on Electronics: "Roo's First Circuit Adventure"
- **Target Age**: 6-8 years.
- **Length**: 24 pages (including cover).
- **Structure**:
  - **Page 1 (Cover)**: Title, Roo holding a flashlight.
  - **Pages 2-3**: What is electricity? (Zap! Lightning, then safe battery).
  - **Pages 4-5**: Conductors vs. insulators (metal spoon vs. rubber glove).
  - **Pages 6-7**: Building a simple circuit (battery, wire, bulb).
  - **Pages 8-9**: Adding a switch.
  - **Pages 10-11**: Series vs. parallel (two bulbs).
  - **Pages 12-13**: Fun project: Paper clip switch.
  - **Pages 14-15**: Troubleshooting: Why isn't it light? (loose wire, dead battery).
  - **Pages 16-17**: Safety note: Ask an adult, only use batteries.
  - **Pages 18-19**: Quiz: Match the symbol to the part.
  - **Pages 20-21**: Draw your own circuit.
  - **Pages 22-23**: Roo's Challenge: Make a lighthouse.
  - **Page 24 (Back)**: About Roo, next book teaser (magnets).
- **Features**:
  - Big, colorful illustrations.
  - Simple sentences, 1-2 per page.
  - Interactive elements: "Can you spot the broken wire?"
  - Includes a cut-out template for a paper circuit on the last page (or separate sheet).
- **Next Steps**:
  1. Outline each page with text and sketch notes.
  2. Create illustrations (can use simple vector art).
  3. Write and review for clarity.
  4. Format for print/PDF (8.5"x8.5").
  5. Publish via Amazon KDP or offer as free PDF to subscribers.
